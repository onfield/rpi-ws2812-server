This command makes green text message scroll.
Connect a 8x32 LED panel to PWM0 output.

setup 1,256,3 init brightness 1,32 config_2D 1,32,8,3,32,8 init_layer 1,0,1,0,1,0 message_board 1,0,8,32,8,0,00FF00,000000,50,0,"HELLO WORLD",8,,0,"enhanced_led_board-7.ttf"

NOTE: Put double quotes around your text, use \" if you need a double quote in your text. For special characters use UTF-8 encoding and a font that supports them.

If you want to print message on a background image like in the youtube example. You need 2 extra layers for that, one you render the background image (red-green.png) and the other the text is rendered, then both are combined and only where there is text the background image will be visible and rendered on the bottom layer Use code:
setup 1,256,3 init brightness 1, 32 config_2D 1,32,8,3,32,8 init_layer 1,1,1,0,1,0 change_layer 1,1 draw_image 1,examples/red-green.png init_layer 1,2,8,0,1,0 change_layer 1,2 message_board 1,0,8,32,8,0,FFFFFF,,50,0,"HELLO WORLD",8,,0,"enhanced_led_board-7.ttf"

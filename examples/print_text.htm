This command prints text on an 8x32 display.

setup 1,256,3 init brightness 1,32 config_2D 1,32,8,3,32,8 init_layer 1,0,1,0,1,0 debug 1 print_text 1,0,8,"HELLO WORLD", FF0000,8,1,0,"enhanced_led_board-7.ttf" render

NOTE: Put double quotes around your text, use \" if you need a double quote in your text. For special characters use UTF-8 encoding and a font that supports them.

The next example will print text with gradient colored letters, using extra layer and a background picture.
setup 1,256,3 init brightness 1, 32 config_2D 1,32,8,3,32,8 draw_image 1,examples/red-green.png init_layer 1,1,8,0,1,0 change_layer 1,1 print_text 1,0,8,"HELLO", FF0000,8,1,0,"enhanced_led_board-7.ttf" render
